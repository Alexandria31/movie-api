Movie API Documentation

This API allows you to manage a list of movies, providing endpoints for creating, retrieving, updating, and deleting movie records.

Base URL

http://localhost:3000

Endpoints

1. Get All Movies

Method: GET

Endpoint: /movies

Description: Retrieve a list of all movies.

GET /movies

2. Get a Movie by ID

Method: GET

Endpoint: /movies/:id

Description: Retrieve a specific movie by its ID.

GET /movies/1

3. Add a New Movie

Method: POST

Endpoint: /movies

Description: Add a new movie to the list.

Request Body:

{
"title": "Interstellar",
"director": "Christopher Nolan",
"year": 2014
}

4. Update a Movie

Method: PUT

Endpoint: /movies/:id

Description: Update an existing movie by its ID.

Request Body:

{
"title": "Inception (Updated)",
"director": "Christopher Nolan",
"year": 2010
}

5. Delete a Movie

Method: DELETE

Endpoint: /movies/:id

Description: Delete a movie by its ID.

DELETE /movies/1

Response Codes

  • 200 OK: Successful request
  • 201 Created: New resource has been created
  • 204 No Content: Successful deletion
  • 404 Not Found: Resource not found

Notes

Make sure to validate the input data when implementing a real API.
