Movie API Documentation

Base URL

http://localhost:3000

Endpoints

1. Movies

Manage movies in the database.

Get All Movies

GET /movies

Returns a list of all movies.

Response:

[
{
"id": 1,
"title": "Inception",
"director": "Christopher Nolan",
"year": 2010,
"genre": "Sci-Fi",
"directorDOB": "1970-07-30"
},
{
"id": 2,
"title": "The Matrix",
"director": "Lana Wachowski, Lilly Wachowski",
"year": 1999,
"genre": "Action",
"directorDOB": "1965-06-21"
}
]

Get a Specific Movie

GET /movies/:id

Returns details of a specific movie by ID.

Response:

{
"id": 1,
"title": "Inception",
"director": "Christopher Nolan",
"year": 2010,
"genre": "Sci-Fi",
"directorDOB": "1970-07-30"
}

404 Error: Movie not found if the ID does not exist.

Get Movies by Genre

GET /movies/genre/:genre

Returns all movies that belong to a specific genre.

Response:

[
{
"id": 1,
"title": "Inception",
"director": "Christopher Nolan",
"year": 2010,
"genre": "Sci-Fi",
"directorDOB": "1970-07-30"
}
]

404 Error: No movies found for the specified genre.

Get Movies by Director

GET /movies/director/:director

Returns all movies directed by a specific director.

Response:

[
{
"id": 1,
"title": "Inception",
"director": "Christopher Nolan",
"year": 2010,
"genre": "Sci-Fi",
"directorDOB": "1970-07-30"
},
{
"id": 3,
"title": "Interstellar",
"director": "Christopher Nolan",
"year": 2014,
"genre": "Adventure",
"directorDOB": "1970-07-30"
}
]

404 Error: No movies found for the specified director.

2. Users

Manage users in the database.

Get User by ID

GET /users/:id

Returns details of a specific user by ID.

Response:

{
"username": "john_doe",
"dob": "1990-01-01",
"email": "john_doe@example.com",
"password": "securepassword123",
"favoriteMovies": ["Inception", "The Matrix", "Interstellar"]
}

404 Error: User not found if the ID does not exist.

Add a New User

POST /users

Creates a new user with the following body:

{
"username": "new_user",
"dob": "1995-05-05",
"email": "new_user@example.com",
"password": "newpassword123"
}

Response:

{
"username": "new_user",
"dob": "1995-05-05",
"email": "new_user@example.com",
"password": "newpassword123",
"favoriteMovies": []
}

Update User

PUT /users/:id

Updates details of a specific user by ID with the following body:

{
"username": "updated_user",
"dob": "1990-01-01",
"email": "updated_user@example.com",
"password": "updatedpassword123"
}

Response: Returns updated user details.

Deregister User

DELETE /users/:id

Removes a user from the database by ID.

Response:

"User john_doe has been deregistered."

404 Error: User not found if the ID does not exist.

Add a Movie to Favorites

POST /users/:id/favorites

Adds a movie to the user's favorites with the following body:

{
"movie": "Inception"
}

Response: Confirmation message.

Remove a Movie from Favorites

DELETE /users/:id/favorites

Removes a movie from the user's favorites with the following body:

{
"movie": "Inception"
}

Response: Confirmation message.

Default Route

GET /

Returns a welcome message.

Response: "Welcome to the Movie API"

Notes

Ensure to replace :id, :genre, and :director with the actual user ID, genre, or director name when making requests.
